🚀 Team Task Management SaaS

A full-stack Task Management application designed for teams to collaborate efficiently.

------------------------------------------------------------

✨ FEATURES

- 📊 Project-Based Dashboard
- 📁 Project Management
- 🛡️ Role-Based Access (Project-specific)
- ✅ Task Management
- 📊 Role-Based Dashboard Views
- 🎨 Clean UI/UX

------------------------------------------------------------

🛠️ TECH STACK

Frontend: React (Vite), Tailwind CSS  
Backend: FastAPI, SQLAlchemy  
Database: PostgreSQL (Supabase)  
Deployment: Railway  

------------------------------------------------------------

🚀 GETTING STARTED

Clone Repo:
git clone <repository-url>

Backend:
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

Frontend:
cd frontend
npm install
npm run dev

------------------------------------------------------------

📖 API DOCS
http://localhost:8000/docs

------------------------------------------------------------

🚀 DEPLOYMENT

Backend + Frontend deployed on Railway
